# win_vimrc

windows用vim 設定ファイル

## Install

1. gitをインストールし、パスを通す
git for windows インストーラー
https://git-for-windows.github.io/


2. vimをインストール
香り屋版 Vim 最新リリース情報
https://github.com/koron/vim-kaoriya/releases/latest

3. 本リポジトリダウンロード
git clone https://github.com/rmatttu/win_vimrc.git

4. install.batを起動
